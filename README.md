<details>

</details>

<details>
  <summary>🌟 威士忌酸 (Whiskey Sour)</summary>
    - 60ml 威士忌

    - 1 个蛋清

</details>

    - 15ml 糖浆  
    - 60ml 苏打水补满  
  - **调制方法**：先摇匀 (Shake)，再搅拌 (Stir)
